## 🚀 Technologies Used

### 🧩 Backend (Spring Boot)
- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA
- H2
- Maven

### 🎨 Frontend (React.js)
- React 18+
- Axios (for API calls)
- React Router
- Bootstrap / Tailwind CSS (customizable)
